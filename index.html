We want corners!

A browser extension that sharpens rounded corners on web pages.

Web pages used to be built with boxes. Boxes with corners so sharp you could cut your finger on them. But increasingly, web designers are eschewing this angular order. Instead they are lured by the siren-call of the 'squircle'.

⮕

We Want Corners is a stand against this onslaught of babyproofed web pages. Banish the tyranny of the squircle and restore crisp, beautiful angles to your browsing experience!

FAQs

  • How does it work?

    The extension resets the border-radius CSS rule to 0 using a wildcard selector, like this:

    * { border-radius: 0 !important }

    That's it. That's the whole extension.

  • Can I limit WWC to certain sites?

    Yes. You can choose only to run WWC on certain sites using your browser preferences, or you can run it all the time and exclude certain sites by clicking the WWC item your browser toolbar.

  • Can’t I just do this with Stylebot or Stylish?

    Sure, but WWC is more a badge of honor than anything. Plus, it's super lightweight and won't make you sick, kill you, or spy on your browsing history.

  • Is WWC open source?

    Yes, you can find it on Github.

  • Why am I still seeing some rounded corners with the extension installed?

    WWC is pretty basic. It can't square off things like images or SVG's, and it can be overruled by a more specific CSS selector. If you have an improvement, please open a pull request.

  • Why are innocent circles being converted squares?

    Web developers use the same CSS rule to make circles that they use to round the corners of rectangles. It's not easy to know which one the developer was going for, so circles become squares.

  • I’ve noticed a bug, what should I do?

    Please open an issue on the Github repo issues page.
